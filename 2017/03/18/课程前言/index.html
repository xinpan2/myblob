课程前言

发表于 2017-03-18   |   分类于 R   |  

相信大家在学习R之前，都听说过R语言的用途，比如数据分析、数据可视化等等，之前我就听说过用MATLAB来进行数据分析，但是它是商业版的，并且也很“笨重”，但是R是开源的。

我们先来看看，数据挖掘的层次模型：

从下往上的顺序：

  • DataSources：代表源数据，即从某个接口中获取到的数据，这时的数据可能包含无效数据。
  • DataWarehourses：代表数据仓库，此时的数据经过ETL（数据的处理：提取、转换、加载），被存储到仓库中，ETL过程我们在学习hadoop时，用过MR程序来处理，此时的数据是根据需求过滤得到的结果。
  • DataExploration：数据操作，这里的数据操作指对数据进行统计学上的中位数、众数、求均值等运算。
  • DataMining： 这里进行数据的探索，使用的方法是不确定的，不像求数据的众数，均值那样，这里要处理的是从一堆数据中，发现从肉眼看不出来的数据间的关联，使用到众多算法，比如贝叶斯算法、K-means、神经网络等等，这些在后面会讲到。
  • DataPresentation：数据可视化，在传统的数据可视化，我们只是将处理完的数据，在Excel中展示出来，但是我们通过R可以把这些数据用图表输出，让结果更加一目了然。
  • MakingDecisions：把可视化的数据交给决策者进行决策。

在我们学习R的内容中，包含三个部分，从数据操作->数据可视化。

当然，用来进行数据分析的工具有很多，除了R之外，比较出名的还有sas、spss。当然，甲骨文和DB2中也有数据分析函数，但是他们对于有些算法的测试需要编写长篇大论的sql语句来实现，但是其他工具，比如R、sas或者spss只要一句函数即可实现，就像当初用java调用支付接口时，编写的代码非常臃肿。

R介绍

下面几段话是引用官网对R的介绍：

Introduction to R

R is a language and environment for statistical computing and graphics. It is a GNU project which is similar to the S language and environment which was developed at Bell Laboratories (formerly AT&T, now Lucent Technologies) by John Chambers and colleagues. R can be considered as a different implementation of S. There are some important differences, but much code written for S runs unaltered under R.

R provides a wide variety of statistical (linear and nonlinear modelling, classical statistical tests, time-series analysis, classification, clustering, …) and graphical techniques, and is highly extensible. The S language is often the vehicle of choice for research in statistical methodology, and R provides an Open Source route to participation in that activity.

One of R’s strengths is the ease with which well-designed publication-quality plots can be produced, including mathematical symbols and formulae where needed. Great care has been taken over the defaults for the minor design choices in graphics, but the user retains full control.

R is available as Free Software under the terms of the Free Software Foundation’s GNU General Public License in source code form. It compiles and runs on a wide variety of UNIX platforms and similar systems (including FreeBSD and Linux), Windows and MacOS.

大意：

R是进行统计计算和图形展示的语言和环境，它属于GNU项目，同时与S语言（在贝尔实验室被John Chambers和其他同事发明）相似。我们也可以认为R是S的另外一种实现，虽然他们之间有一些重要的区别，但是大部分在S中能运行的代码,在R中也能运行。

R为我们提供了非常多种高可扩展的统计学和图形学技术。S语言作为统计方法研究的首选工具，R为参与到统计方法研究提供了一条开源的路径。

R的其中一个好处是在需要时可以很好的设计出和发表自己的创意，包括数学符号和公式。对于图形中的次要选项的默认值需要非常细心，但是为用户提供完全控制。

R是一个开源、免费的GNU软件，它可以在许多不同的操作系统中进行编译和运行。
