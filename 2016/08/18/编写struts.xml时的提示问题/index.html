编写struts.xml时的提示问题

发表于 2016-08-18   |   分类于 struts入门   |  

版权声明：本站采用开放的[知识共享署名-非商业性使用-相同方式共享 许可协议]进行许可

所有文章出现的代码，将会出现在我的github中，名字可以根据类全名来找，我在github中的文件夹也会加目录备注。

1、联网即可解决，因为编译器（当前使用myEclipse会根据struts.xml中的引如dtd的地址，自动访问相应的网页并加载内容）

2、没网：

  • 找到struts.xml中的引如dtd说明那部分

把http://struts.apache.org/dtds/struts-2.1.7.dtd拷贝下来当作备用

  • 在各自下载的strtus压缩包中搜索*.dtd， 然后把我们当前用的版本对应的dtd文件复制到当前工作目录下（workspace）

  • 在window选项中选择preference—〉然后搜索xml，通过显示结果选择 xml catalog

  • 点击add按钮，此时location栏中，通过选择workspace按钮，选择刚刚拷贝到工作空间的dtd文件

  • key type栏中选择Uri

  • Key把刚刚拷贝的dtd地址复制上去

  • 此时打开struts.xml试一下有没有提示

  • 如果没有，把struts.xml关闭了再打开即可
