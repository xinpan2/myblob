创建configuration对象

Configuration configuration=new Configuration();

创建配置文件对象，这里使用默认方式加载配置文件

configuration.configure();

配置文件中存放的信息有：

  • 映射文件信息（包括pojo及其属性，系统就是根据映射文件的class属性来认定哪些类为pojo的）

  • 数据库的连接信息

得到sessionFactory对象

SessionFactory sessionFactory=configuration.buildSessionFactory();

note that:

  • 因为一般的配置文件中只出现一个sessionFactory，所以sessionFactory是由单例模式产生的

  • 由于该类中含有许多信息，所以该类是一个重量级类，并且其中的内容是共享的，所以该类是线程安全的

得到session对象,准备操作数据库

Session session=sessionFactory.openSession();

如果只进行查询操作，不用开启事务，直接操作

如果进行增删改操作，要开启事务

Transaction transaction=session.beginTransaction();

然后操作完成后提交事务

transaction.commit();

关闭session

session.close();
